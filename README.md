# Nuxt 3 프로젝트

Nuxt 3를 사용한 프로젝트입니다.

## 설치

```bash
npm install
```

## 개발 서버

개발 서버를 `http://localhost:3000`에서 시작합니다:

```bash
npm run dev
```

## 프로덕션 빌드

프로덕션용 애플리케이션 빌드:

```bash
npm run build
```

프로덕션 빌드 미리보기:

```bash
npm run preview
```
